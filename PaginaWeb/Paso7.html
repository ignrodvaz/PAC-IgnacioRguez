PASOS EN UBUNTU

Cuando está instalado, como root editamos con vim el archivo de configuración:

Al escribir el siguiente comando:
Vim /etc/samba/smb.conf -> si este comando no funciona prueba con "Nano" en vez de "Vim"

Dentro de Vim vamos al final y escribimos al final del documento (tecla Insert)

  • [Compartida]
    path=/home/winuser/compartida
    browseable=yes
    readonly=no
    validusers = winuser

Saldremos de Vim pulsando escape y escribiendo :wq -> (write, quit)

  1. Cambiamos a root
  2. Buscamos actualizaciones
  3. Instalación Samba
  4. Creacion nuevo usuario
  5. Creacion carpeta "Compartida"
  6. Instalamos Vim
  7. Configuramos el archivo
  8. Añadimos "Winuser" a Samba
  9. Cambiamos a "Winuser"
  10. Reseteo samba desde root
  11. Miramos la direccion IP
  12. Hacemos ping a windows
